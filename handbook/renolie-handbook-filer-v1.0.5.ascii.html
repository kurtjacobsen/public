RENOLIE Handbook (filer) v1.0.5 (ascii)

Gyldig fra: 2025-10-26 · Tidszone: Europe/Copenhagen

0) Formaal

Dokumentet erstatter v1.0.3 i dette projekt og samler opdateringer for M1→M2.

1) Leveringsregler

  1. Faerdige downloads: Filer leveres som direkte downloadbare filer. Kode listes ikke i traaden. Ankertekst = eksakt filnavn.
  2. ZIP=off (kun ved eksplicit anmodning). README=off (kun ved klart behov).

2) Filnavngivning (frontend/js)

  • Strengt format for js: [a-z][0-9]{3}.js (fx r101.js, f104.js, n101.js).
  • Testfiler: brug x-serien (x001.js, x010.js ...).
  • Ingen ?v=-parametre i fil-URL'er.

3) M1 (status)

  • r101 (rules.core), f104 (ui.format), n101 (net.client), a101 (stream-adapter) + strict integrationstest x010.js = GRØN.

4) M2 (maal)

  • s101: persona save/load/remove (mock nu; senere Redis).
  • c101: allowlist load/parse (mock nu; senere HTTP via n101).
  • u102: UI-controller, som binder s101 + c101 + f104 (+ n101 live senere).

5) ASCII/formatteringspolitik

  • Transport/lagring: ren ASCII.
  • UI: danske bogstaver som HTML-entities (&aelig; &oslash; &aring; ...).
  • PDF: UTF-8 via font-URL (uaendret fra v1.0.3).

6) Proxy/Render baseline

  • Build: npm install --omit=dev · Start: node renolie-proxy.mjs
  • Health: /healthz viser OK + enc-check.
  • CORS_ORIGIN: https://shop96018.sfstatic.io, https://www.renolie.dk, https://renolie.dk

7) Status/Traadbelastning

  • Hvert svar starter med: Traadbelastning: __ % (0–100).
  • Advar ved 70% og foreslaa traadskifte.

8) DoD (kort)

  • Kontrakt-header, 5–8 kontrakt-tests pr. modul, ingen dybe afhaengigheder, changelog ≤ 10 linjer.

9) Henvisninger

  • M1 harness (browser): renolie-m1-harness.v2.ascii.html + w101.js (mock-only).
